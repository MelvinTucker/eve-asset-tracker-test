# EVE Online Asset Tracker

Eve online asset tracking application with Evidence Integrations.

## Connections

- OpenClaw: https://openclawatif
- Google Drive: Google Drive API (validation pending credentials)
- Google Sheets: Google Sheets API v4 (validation requires credentials)
- Gmail: Zapier MCP gmail send (functional)
- Telegram: OpenClaws message tool (functional)
